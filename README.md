# JaBlog
Java RESTful web server. 

Works like a mini twitter.

Backend of the FederatedBirds Android app. https://github.com/acharp/FederatedBirds

## Built with
- Java 8 
- Google App Engine 
- Google DataStore and Objectify library to store simple datas (users, messages, followings, ...)
- Google Cloud Storage to store files (e.g. profile pictures)
- GSON

If you just want to test the web service, it is actually hosted at https://jablog-1158.appspot.com/
Send requests with right formated json and see the responses.
